# SaveTab.csv

**SaveTab.csv** is a Chrome Extension that allows users to save open tabs locally with rich metadata (Categories, Tags, Scrollable Screenshots) and export them to a CSV file.

## Features
- **Local Storage**: All data is saved to your local file system using the File System Access API.
- **Rich Metadata**: Save tabs with Categories and Tags for easy organization.
- **Screenshot Capture**: Automatically capture and stitch full-page screenshots of your tabs.
- **Global Viewer**: View, filter, search, and manage your saved tabs in a dedicated dashboard.
- **CSV Export**: Instantly download your database as a standard `.csv` file.
- **Customizable**: Dark/Light mode, customizable screenshot depth.

## Installation
1. Clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select the directory containing this repository.
